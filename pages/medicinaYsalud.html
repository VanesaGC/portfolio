Tema del proyecto

Se trata de una aplicación web en ASP.NET, dónde el ciudadano tinerfeño, podrá consultar cualquier centro médico, farmacia, clínicas dentales y centros de fisioterapeutas y rehabilitación de todo Tenerife. Además estos, centros los podrá filtrar por actividad y por municipio. Pudiendo también ver estos sitios, situados en un mapa para mayor comodidad del usuario. Para poder acceder a esta herramienta, los usuarios tendrán que registrarse previamente de manera gratuita.

Tipos de usuarios

La página web tiene tres tipos de usuarios: anónimo, registrado y administrado. Cada tipo de usuarios tienen permisos con respecto al uso de los datos de la página web.

Información técnica

Para la realización del proyecto, se han usado las siguientes tecnologías: ASP.NET, Entity Data Model, MVC5, Google Maps API, y servicio web.
