Tema del proyecto

El objetivo de este trabajo, era crear un buscador de farmacias de Tenerife en una página web. De forma que el ciudadano o turista pudiera buscar y ubicar, de una manera cómoda y sencilla las farmacias más cercanas.

En la realización del proyecto, se tuvo en cuenta el mantenimiento de la página. Ya que, las farmacias cambian de nombre, ubicación o incluso cierran.

Para mostrar al usuario de una forma rápida y visual los datos, creé un gráfico con los porcentajes de farmacias por municipio. Debajo del gráfico se puede ver una tabla paginada con los porcentajes.

Tipos de usuarios

La página web tiene tres tipos de usuarios: anónimo, registrado y administrado. Cada tipo de usuarios tienen permisos con respecto al uso de los datos de la página web.

Mapa

El mapa introducido, está tanto en modo relieve como satélite. Además, de mover el muñeco situado a la izquierda para ver con exactitud, dónde está situada la farmacia encontrada.

En los cuadros de dialogos de los marcadores de las farmacias comentadas con imágenes, aparecerá la última imagen subida sobre esa misma farmacia.

Información técnica

Para la realización del proyecto, se han utilizado las siguientes tecnologías: PHP, API Google, JavaScript, HTML5 y CSS.
